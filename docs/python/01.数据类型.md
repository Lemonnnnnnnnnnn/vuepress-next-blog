## 数据类型

### 字符串

```python
print(r'\t') # \t  r''表示不转义
print('\t') # 

print('''
        1
        2
        3
      ''') # 用''' 表示多行内容代替\n

s = 'abc'
s2 = s.replace('a' , 'A') # 替代
```

### 元组

元组 `tuple`和列表 `list` 非常类似，唯一的区别是它一旦初始化就不能修改。

优点：使用元组可以让你的程序更安全。

```python
t = (2, 3) 
t = (2,) # 为了区分元素和运算符括号，在声明只有一个元素的元组时会在后面加一个逗号
```

### 字典

```python
d = {foo : 'foo'} # 定义
print('bar' in d) # 是否存在
print (d.get('bar') == None)   # 是否存在
d.pop('foo') # 删除 

```

### set

```python
s = {"1","2"}
s = set([1,2,3])
s.add(4)
s.remove(4)
s2 = set([2,3,4])
print(s1 & s2) # 利用位运算符 & 取交集
print(s1 | s2) # 利用位运算符 | 取并集

# a & b ：将a和b转化为二进制，将每一位 用& 计算
# a | b : 将每一位 用| 计算

```
