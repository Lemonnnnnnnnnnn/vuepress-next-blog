## mysql workbench

#### 快捷键

-   回退 **ctrl + z**
-   前进 **ctrl + y**
-   执行所有语句 ： **ctrl + shift + enter**
-   执行当前光标所在行： **ctrl + enter**

## mysql command line

#### 多行命令时如何取消/返回修改前边的命令

-  **\c** : 重新输入
-  **\p** : 打印前面输入的行
