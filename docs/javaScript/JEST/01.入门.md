## 安装

```bash
yarn add --dev jest
```

## vscode中jest代码智能提示

```bash
npm i -D @types/jest
```

::: tip
vscode只会读取 **根目录的node_modules** 中的@types库
:::

## 起步案例

`sum.js`:

```js
function sum(a, b) {
  return a + b;
}
module.exports = sum;
```

`sum.test.js` :

```js
const sum = require('./sum');

test('adds 1 + 2 to equal 3', () => {
  expect(sum(1, 2)).toBe(3);
});
```

`package.json` : 

```js
{
  "scripts": {
    "test": "jest"
  }
}
```

执行`yarn test`:

```bash
PASS  ./sum.test.js
✓ adds 1 + 2 to equal 3 (5ms)
```
